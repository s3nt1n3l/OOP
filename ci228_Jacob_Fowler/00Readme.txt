To run Catshop

1) Go to folder "class"
2) Double click setup.jar
2) Double click main.jar

Catshop will now open!

Alternatively...

To run from Cmd

1) Navigate to folder "class"
2) Type "java -jar setup.jar" and hit enter
3) wait for it to finish setting up
4) Type "java -jar main.jar" and hit enter

Catshop will now open!
